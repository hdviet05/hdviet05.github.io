Matrix - Determinant (continued)

03.10.2023

Properties

Let matrices \(A, A' \) having size \(m \times n\), \(B and B\) having size \( n \times p\), \( C\) having size \( p \times s\), and real number \( \alpha\)

  • i. \((AB)C = A(BC) \)
  • ii. \((A+A')B = AB + A'B\)
  • \(A(B+B') = AB + AB' \)
  • iii. \((2 \alpha ) B = A( \alpha B) = \alpha (AB) \)
  • iv. Let \(E_n\) and \(E_m \) be identity matrices having size \( n \times n\) and \(m \times m \), respectively. Then \( AE_n = A \), \(E_mA = A \). Especially, if A is a square matrix of size \(n\), then \(AE_n = E_nA = A \).

Remarks

Matrix multiplication is not commutative.

Definition

Matrices \(A \) and \( B \) are said to commute if \( AB = BA \). (They ought to be square matrices of the same size in advanced).

Example
