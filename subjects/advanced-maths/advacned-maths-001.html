1.1. Basic concepts of the matrix

1.1.1. Definition

A matrix of size m x n is a rectangular array of numbers, arranged in m rows and n columns

A matrix is often denoted by capitalized letters like A, B, C,...

$$A = \begin{bmatrix} a_{11} & a_{12} & ... & a_{1n} \\ a_{21} & a_{22} & ... & a_{2n} \\ a_{m1} & a_{m2} & ... & a_{mn} \end{bmatrix}$$

It can also be written as:

$$A = [a_{ij}]_{mn}$$

Or:

$$A = (a_{ij})_{mn}$$

\(a_{ij}\) is the element in the \(i^{th}\) row and \(j^{th}\) column of \(A\).


The Zero Matrix

is the matrix whose all elements are 0.

is denoted by \((O)_{mxn}\), or \((O)_{mxn}\).


Equality of two matrices

Let \(A = [a_{ij}]_{mxn}\), \(B = [b_{ij}]_{mxn}\)

\(A = B \iff a_{ij} = b_{ij}, \forall i = 1, 2, ...,m; j = 1, 2, ..., n.\)


The Transpose of a matrix

The transpose of a matrix \(A = [a_{ij}]_{mxn}\) is found by interchanging its rows into columns and vice versa.

The tranpose of \(A\) is denoted by \(A^T\).

$$ A^T = [a_{ij}]_{nxm} $$


1.1.2. Square Matrices

A square matrix is a matrix with the same number of rows and columns.

$$A = \begin{bmatrix} a_{11} & a_{12} & ... & a_{1n} \\ a_{21} & a_{22} & ... & a_{2n} \\ a_{n1} & a_{n2} & ... & a{nn} \end{bmatrix}$$


$$ A = [a_{ij}]_n $$


Let \(A = [a_{ij}]_n \) be an n-square matrix. The main diagonal of \(A\) consists of the elements \(a_{11}, a_{22}, ..., a_{nn} \).

The trace of \(A\), denoted by \(Tr(A )\), is the sum of the main diagonal elements.

That is:

$$Tr(A) = a_{11} + a_{22} + ... + a_{nn} $$


Special Square Matrices

Let \(A = [a_{ij}]_n \) be a square matrix.


Diagonal Matrices

\(A\) is called a diagonal matrix if all elements outside of the main diagonal of \(A\) are zero.

$$ a_{ij} = 0, \forall i \neq j $$

\( A = \begin{bmatrix} 11 & 0 & 0 \\ 0 & -5 & 0 \\ 0 & 0 & 6 \end{bmatrix} \) is a 3 x 3 diagonal matrix.


Identity Matrices

A is called an identity matrix if A is the diagonal matrix with \(a_{ij} = 1, \forall i = 1, 2, ..., n \).

$$ A = \begin{bmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{bmatrix} \begin{cases} a_{ij} = 1, \forall i = \overline{1, n} \\ a_{ij} = 0, \forall i \neq j \end{cases} $$


Upper Triangular Matrices

\(A\) is called the upper triangular matrix if all elements below the main diagonal are zero.

$$a_{ij} = 0, \forall i > j $$

$$A = \begin{bmatrix} -3 & 7 & 10 \\ 0 & \frac{1}{2} & 15 \\ 0 & 0 & 5 \end{bmatrix}$$


Lower Triangular Matrices

\(A\) is called the lower triangular matrix if all elements above the main diagonal are zero.


Symmetric Matrices

\(A\) is called a symmetric matrix if \(A\) equals its transpose.

$$A = A^T $$

$$ a_{ij} = a_{ji}, \forall{i, j = 1, 2, ..., n.} $$


1.2. Basic Operations of Matrices

1.2.1. Matrix Addition

Definition

The sum of two matrices \(A = [a_{ij}]_{m\times n} \) and \(B = [b_{ij}]_{m \times n} \), denoted by \(A + B \), is a matrix which has the same size as \(A\) and \(B\) with its \((i, j) \) element computed by adding \(a_{ij}\) and \(b_{ij}\).


Properties

Let \(A = [a_{ij}]_{m \times n} \), \(B = [b_{ij}]_{m \times n} \) and \(C = [c_{ij}]_{m \times n} \)

$$ \rightarrow A+B=B+A$$

$$ A+(B+C)=(A+B)+C$$

$$ A+(O)_{m \times n} = A$$


1.2.2 Multiplication of matrices by real numbers (scalar multiplication).

Definition

Let there be matrix \(A=[a_{ij}]_{m \times n} \) and a real number \(k\). The multiplication of \(A\) with \(k\), denoted by \(kA\), is a matrix which is obtained by multiplying each element of \(A\) by \(k\).


Properties

Let \(A=[a_{ij}]_{m \times n}, B=[b_{ij}]_{m \times n}\) and real numbers \(\alpha , \beta \):

$$ \alpha (\beta A) = \beta (\alpha A) = (\alpha \beta ) A$$

$$ \alpha (A+B) = \alpha A + \alpha B$$

$$ (\alpha + \beta)A = \alpha A + \beta A$$

$$ 1 A = A$$

$$0 A = (O)_{m \times n}$$


1.2.3. Product of Matrices

Definition

The product of the matrices \(A=[a_{ij}]_{m \times n} \) and \(B=[b_{ij}]_{m \times n} \) is a matrix having size \(m \times n\), denoted by \(A B = [c_{ij}]_{m \times n} \), with \(c_{ij}\) defined as \(\sum_{s=1}^k a_{is} \times b_{sj}, \forall i = \overline{1, m} , j = \overline{1,n} \).


Remarks

The product \(AB\) is defined if and only if the number of columns in \(A\) equals the number of rows in \(B\).

Matrix \(AB\) has the same number of rows as \(A\) nd the same number of columnns as \(B\).
